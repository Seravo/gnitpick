# Gnitpick

Git commit message [nitpicker](https://en.wiktionary.org/wiki/nitpick). Makes sure all future git commit messages are perfect!

## Usage

In you git pre-commit hook or CI system, simply run:

    python3 /path/to/gnitpick.py
